Welcome

Добро пожаловать на сайт Survyes. Это сайт для прохождения опросов онлайн. Если у вас есть какие-нибудь интересные идеи для создания собственного опроса, вы можете предложить его нам, но для этого вам необходимо зарегистрироваться и войти в личный кабинет. Если вы уже зарегистрированный пользователь, просто зайдите в личный кабинет и нажмите создать опрос.

Кто может проходить опросы?

Любой посититель сайта может проходить опросы, просто перейдя по вкладке Topics на навигационной панели, но проходя опросы без входа в систему, результаты прохождения не будут записаны.
